Pattern library for regjeringen.no

This is a resource for people working with the Government website regjeringen.no. It aims to provide an overview of typical elements and components in the user interface of the website.

The pattern library can be used to maintain modular frontend code and to ensure a visual consistency on the website of the government.

									$("article table").tablewrapper({

									});

Wrap wide content plugin (replacement for the table-wrapper plugin above)

This plugin will hide overflowing parts of tables which is too wide for its surrounding container (parent-element) and add a button above the table which (when clicked) opens the table in a lightbox. The plugin depends on script lightbox-content.js (jQuery dependent):

							
								<script src="/dist-web/js/lightbox-content.js"></script>
								<script>
									$('.article-body img, .article-body table').wrapWideContent({
										//container: $('.article-body, .longdoc-content'),
										//threshold: 1.7856,
										langZoomImage: 'Forstørr bilde',
										langShowFullTable: 'Vis tabellen i full bredde'
									});
								</script>

    • Integrering

      Subtittel

      Hovudmålet for integreringspolitikken er at innvandrarar og borna deira skal få bruke ressursane sine og bidra til fellesskapet. Nøklane til integrering i det norske samfunnet ligg i å delta i arbeidslivet og ha gode norskkunnskapar.

    • Stortingsmelding om humaniora

      Regjeringen starter nå arbeidet med en egen melding til Stortinget om humanistisk forskning og utdanning og tar sikte på å legge den frem i 2017.

      Sitatblokk

    • Meld. St. 25 (2015–2016)

      Subtittel

      Stortingsmeldingen gir en bred gjennomgang av utviklingstrekk, status og perspektiver for den innenlandske energiforsyningen. Den legger frem en energipolitikk mot 2030, hvor energiforsyning, klimautfordringer og næringsutvikling ses i sammenheng.

    • Informasjonssikkerhet

      Subtittel

      Norge skal være i front med å utvikle og levere digitale tjenester til innbyggere og næringsliv. Et slikt mål hviler også på god informasjonssikkerhet og ivaretagelse av personvernet.

    • Antarktis

      I Antarktis finnes noen av de siste store urørte naturområdene i verden. Planter, dyr, økosystemer og kulturminner i Antarktis er under stadig større press fra ferdsel, miljøgifter og klimaendringer. Samarbeidet under Antarktistraktaten er avgjørende for å sikre at naturmiljø og kulturminner blir bevart.

      Sitatblokk

    • Miljøvern og miljø-utfordringer i nord

      Nordområdene har noe av den flotteste naturen Norge kan by på, og et rikt naturmangfold både på land og i havet. Miljøtilstanden er stort sett god, men klimaendringene kan få dramatiske konsekvenser for alt liv i Arktis. Isavhengige arter som isbjørn er spesielt utsatt. Strenge miljøkrav og beskyttelse av sårbare områder skal sørge for at belastningen ikke blir for stor når aktiviteten øker.

      Sitatblokk
